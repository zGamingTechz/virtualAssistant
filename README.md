# virtualAssistant

just a simple python project
nothing much

Features:
• works fully on speech.
• works in Idle mode and takes voice input only after you say it's name "lisa".
• uses Google speech recognition to recognise your voice and convert it to speech for futher processing.
• uses Microsoft text to speech to deliver output in verbal form.
• can deliver information from Wikipedia on any topic.
• wishes you based on the time.
• can play YouTube.
• can search Google.
• can open Stackoverflow.
• can toss a coin for you.
• can pick a number for you.
• can tell the time.
• can applaud for you!
• Fun little commands and features like information about you and the bot.
